# Install CentOS

## Source

    sudo yum install epel-release

    yum install https://www.elrepo.org/elrepo-release-7.el7.elrepo.noarch.rpm

## Environment

    sudo yum groupinstall "Development Tools" 
  
selinux:

        # SELINUX=enforcing
        SELINUX=disable
        
- 立即生效

        setenforce 0

        getenforce
